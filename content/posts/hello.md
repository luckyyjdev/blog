## 欢迎

这是我的第一篇博客文章。使用 Hugo + PaperMod 主题搭建，部署在 Cloudflare Pages 上。

## 技术栈

- **Hugo**: 静态网站生成器
- **PaperMod**: 简洁优雅的主题
- **Cloudflare Pages**: 免费的静态网站托管

## 本地开发

```bash
hugo server
```

然后访问 `http://localhost:1313` 预览网站。

## 写文章

使用以下命令创建新文章：

```bash
hugo new content posts/my-new-post.md
```

文章默认为草稿状态，修改 `draft: true` 为 `draft: false` 后才会发布。
